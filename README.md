# InternLM-ExcelCopilot
## 介绍
  基于InternLM的智能报表助手
## 功能特性
  智能报表助手 Bot，只需上传 excel 文件就能自动生成用户可能需要的报表。如果自动生成的推荐报表不是用户想要的，可以自己输入想要的报表，就会为你自动生成。
